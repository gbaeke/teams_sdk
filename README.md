# Teams Python LLM Demo

A minimal Microsoft Teams Python bot that uses the OpenAI Responses API, exposes a weather tool backed by Open-Meteo, and renders weather results as Teams Adaptive Cards.

## Project Layout

- `hello-llm-agent/`: Teams Python app.
- `hello-llm-agent/src/main.py`: Teams message handler and OpenAI tool loop.
- `hello-llm-agent/src/weather.py`: Open-Meteo weather lookup.
- `hello-llm-agent/src/weather_card.py`: Adaptive Card rendering for weather results.
- `docs/validated-teams-python-llm-setup.md`: known-working setup and validation steps.

## Quick Start

```bash
cd hello-llm-agent
cp .env.example .env
uv sync
uv run python src/main.py
```

See [the validated setup guide](docs/validated-teams-python-llm-setup.md) for Teams CLI login, sideloading, ngrok, app registration, and package download steps.

## Secrets

Do not commit `.env`, generated app packages, or virtual environments. The app expects runtime settings such as `OPENAI_API_KEY`, `OPENAI_MODEL`, and Teams credentials generated by `teams app create`.
